Principle 2 Deterministic, Isolated, Fast

Bad example

@Test
void getDiscount_ShouldApplyOnWeekends() {
    if (LocalDateTime.now().getDayOfWeek()
        == DayOfWeek.SATURDAY) {
        assertEquals(80.0, service.calculate(100));
    }
}

Good example

@Test
void calculatePrice_ShouldApplyWeekendDiscount() {
    // Arrange
    Clock clock = Clock.fixed(
        SATURDAY_INSTANT,
        ZoneId.systemDefault()
    );
    PricingService service = new PricingService(clock);

    // Act
    double actual = service.calculate(100);

    // Assert
    assertEquals(80.0, actual);
}

If time, randomness, threads, files, or network affect the result, create a seam and control it.
