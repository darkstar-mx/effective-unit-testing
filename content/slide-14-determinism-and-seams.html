Determinism and Seams

@Test
void getDiscount_ShouldApplyOnWeekends() {
    if (LocalDateTime.now().getDayOfWeek()
        == DayOfWeek.SATURDAY) {
        assertEquals(80.0, service.calculate(100));
    }
}
@Test
void calculatePrice_ShouldApplyWeekendDiscount() {
    Clock fixedClock = Clock.fixed(
        SATURDAY_INSTANT,
        ZoneId.systemDefault()
    );

    var service = new PricingService(fixedClock);

    assertEquals(80.0, service.calculate(100));
}

Wrap non-deterministic behavior behind injectable seams.
