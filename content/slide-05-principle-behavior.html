Principle 1 Behavior Over Implementation

@Test
void updateInternalState_ShouldCallPrivateMethod() {
    // implementation-coupled test
}
@Test
void deposit_ShouldIncreaseBalance_WhenAmountIsPositive() {
    Account account = new Account(100);

    account.deposit(50);

    assertEquals(new Account(150), account);
}

A test should survive refactoring if the user-visible behavior is unchanged.
