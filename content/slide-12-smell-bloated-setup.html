Smell 6 Bloated or Global Setup

Bad example

@BeforeEach
void setUp() {
    repo = mock(Repo.class);
    email = mock(Email.class);
    audit = mock(Audit.class);
    service = new Service(repo, email, audit);
    // 50 lines more...
}

Good example

@Test
void updateName_ShouldPersistUser() {
    UserRepo repo = mock(UserRepo.class);
    UserService service = new UserService(repo, emailStub(), auditStub());

    service.updateName(1L, "New Name");

    verify(repo).save(any());
}

Keep setup local unless multiple tests truly share the same meaning, not just the same mechanics.
