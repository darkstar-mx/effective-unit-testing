Meaningless Assertions Signal vs Noise

@Test
void findOrder_ShouldReturnOrder() {
    Order result = service.findOrder(123L);

    assertNotNull(result);
}
@Test
void findOrder_ShouldReturnCorrectOrderDetails() {
    Order expected = new Order(123L, "PENDING", 500);

    Order actual = service.findOrder(123L);

    assertEquals(expected, actual);
}

Why it fails

No verification of actual behavior. Poor failure diagnostics.

Rule

Assertions should explain failures clearly.
