What Makes a Test Valuable?

01Fast

Runs in milliseconds.

02Deterministic

Same result every time.

03Readable

Explains behavior.

04High Signal

Failures are useful.

@Test
void processOrder() {
    var result = service.process(order);

    assertTrue(result != null);
}
@Test
void processOrder_ShouldApplyDiscount_WhenUserIsPremium() {
    User user = new User(Status.PREMIUM);
    Order order = new Order(100);

    var result = service.process(user, order);

    assertEquals(90.0, result.total());
}
