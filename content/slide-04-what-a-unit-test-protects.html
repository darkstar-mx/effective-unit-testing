What a Unit Test Protects

Production design

class OrderService {
    private final TaxPolicy taxPolicy;

    OrderService(TaxPolicy taxPolicy) {
        this.taxPolicy = taxPolicy;
    }

    Money checkout(Order order) {
        return taxPolicy.apply(order.subtotal());
    }
}

Good example

@Test
void checkout_ShouldApplyTaxPolicy() {
    TaxPolicy taxPolicy = subtotal -> subtotal.multiply(1.16);
    OrderService service = new OrderService(taxPolicy);

    Money total = service.checkout(orderWithSubtotal(100));

    assertEquals(money(116), total);
}

Protect this

Business behavior, rules, invariants, and contracts.

Do not protect this

Private flow, temporary structure, or harmless internal refactors.
